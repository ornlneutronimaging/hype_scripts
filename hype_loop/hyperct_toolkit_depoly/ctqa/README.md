ctqa
====

Neutron CT reconstruction quality assessment 

Installing
----------

1. *Requirements*

   ```
   numpy
   scikit-image
   torch
   collections
   ```
2. *Clone or download the repository and get inside:*

    ```Python
    git clone https://github.com/ornlneutronimaging/ctqa.git
    cd ctqa
    pip install .
    ```
3. *Uninstall ctqa*

   ```bash
    rm -r ctqa.egg-info
    rm -r build

    pip uninstall ctqa
    ```
